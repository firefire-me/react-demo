# 项目文档

## 项目概述

这是一个基于 **Next.js 16** 构建的现代化企业级后台管理系统模板，采用最新的 App Router 架构和完整的前端技术栈。

**项目名称**: react-demo  
**版本**: 0.1.0  
**Node 要求**: 22.x

---

| **框架** | Next.js | 16.1.5 |

| **样式** | Tailwind CSS | 3.4.14 |

## 核心功能

### 1. 响应式布局
- **桌面端**: 可折叠侧边栏（宽度 64px / 256px 切换）
- **移动端**: 抽屉式导航（Sheet 组件）
- 自适应断点：`md` (768px)、`lg` (1024px)

### 2. 仪表盘首页 (`/`)
- 4 个统计卡片网格布局
- 总收入、用户数、销售额、当前活跃
- 图表和近期销售占位符区域

### 3. 用户管理 (`/users`)
- 使用 TanStack Query 模拟数据加载
- 表格展示用户信息（ID、姓名、邮箱、角色、状态）
- 状态徽章样式区分：
  - 活跃：`default` 样式
  - 待审核：`outline` 样式
  - 停用：`secondary` 样式

### 4. 登录页面 (`/login`)
- 邮箱 + 密码表单
- Zod 校验：
  - 邮箱格式验证
  - 密码至少 6 个字符
- 模拟登录请求（1 秒延迟）

### 5. 设置页面 (`/settings`)
- 预留页面，待扩展

## UI 组件库

项目使用 **Shadcn/UI** 组件库，基于 Radix UI 构建。

### 可用组件

- `Button` - 按钮（6 种变体 + 4 种尺寸）
- `Card` - 卡片容器
- `Input` - 输入框
- `Label` - 表单标签
- `Badge` - 徽章
- `Table` - 表格
- `Sheet` - 抽屉/侧边面板

### 主题系统

使用 CSS 变量实现亮色/暗色主题切换：

- `--primary` - 主色调
- `--secondary` - 次要色
- `--destructive` - 危险色
- `--muted` - 静音色
- `--accent` - 强调色

## 编码规范

### 组件类型
- 默认使用 **Server Components**（服务端组件）
- 需要交互时使用 `'use client'` 标记为客户端组件

### 类型安全
- 严禁使用 `any`
- 所有数据和函数必须定义明确的 `Interface` 或 `Type`

### 样式规范
- 使用 Tailwind CSS 工具类
- 遵循间距系统（`p-4`, `m-6` 等）
- 颜色使用语义化变量（`bg-primary`, `text-muted-foreground`）

### 路径别名
使用 `@/` 作为根目录别名：
- `@/components`
- `@/lib`
- `@/app`

## 核心文件说明

### `app/layout.tsx`
根布局组件，包含：
- Inter 字体引入
- Providers 包裹（React Query）
- Vercel Speed Insights
- 语言设置为 `zh-CN`

### `components/providers.tsx`
React Query 配置：
- `staleTime`: 60 秒（数据 60 秒内视为新鲜）

### `lib/utils.ts`
- `cn()` - 合并 Tailwind 类名的工具函数（clsx + tailwind-merge）
